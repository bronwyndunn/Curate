# API Endpoints

## HTML API

### Root

- `GET /`

### Users

- `GET /api/users/:id`
- `POST /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`

## JSON API

### Pins

- `GET /api/pins`
- `POST /api/pins`
- `GET /api/pins/:id`
- `DELETE /api/pins/:id`

### Boards
- `GET /api/boards/:id`
- `POST /api/boards`
- `PATCH /api/boards/:id`
- `DELETE /api/boards/:id`
### Users

- `GET /api/user/:id/`
- `GET /api/user/:id/followers`
- `GET /api/user/:id/followees/`

### Tags

- `GET /api/posts/tags`
- `POST /api/posts/:id/tags`
